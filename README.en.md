# 🧮 Calculator

A simple calculator built with **HTML5, CSS3, and JavaScript** that performs basic mathematical operations.

## 🚀 Features

* Add numbers and operators to the display.
* Perform calculations by clicking the **=** button.
* Clear the display using the **C** button.
* Simple and intuitive user interface.

## 🛠️ Technologies

* HTML5
* CSS3
* JavaScript (ES6)

## 📌 How It Works

JavaScript adds click event listeners to all calculator buttons. Each button performs a specific action: inserting values into the display, clearing the current operation, or calculating the result using the `eval()` function.

## ▶️ Getting Started

1. Clone this repository.
2. Open the `index.html` file in your web browser.
3. Use the calculator buttons to perform calculations.
