# 🧮 Calculator

Calculadora desenvolvida com **HTML5, CSS3 e JavaScript**, capaz de realizar operações matemáticas básicas.

## 🚀 Funcionalidades

* Adicionar números e operadores ao visor.
* Realizar cálculos ao clicar em **=**.
* Limpar o visor com o botão **C**.
* Interface simples e intuitiva.

## 🛠️ Tecnologias

* HTML5
* CSS3
* JavaScript (ES6)

## 📌 Como funciona

O JavaScript adiciona eventos de clique a todos os botões da calculadora. Cada botão executa uma ação específica: inserir valores no visor, limpar a operação ou calcular o resultado da expressão utilizando a função `eval()`.

## ▶️ Como executar

1. Clone este repositório.
2. Abra o arquivo `index.html` no navegador.
3. Utilize os botões da calculadora para realizar os cálculos.
